# Brand Logos

This directory contains brand logos for the "Brands We Install" section.

## Required Logo Files:
- gaf.png
- certainteed.png
- malarkey.png
- owens-corning.png
- tamko.png
- stoneworth.png
- davinci.png
- versico.png
- mulehide.png

## Recommended Logo Specifications:
- Format: PNG with transparent background
- Dimensions: 300-400px width (maintain aspect ratio)
- Resolution: 72-144 DPI
- File size: Under 100KB each

## How to Obtain Logos:
1. Visit each manufacturer's website press/media kit page
2. Download official logos (usually available in their "Press" or "Resources" section)
3. Or use a tool like https://www.brandeps.com/ to find vector logos
4. Ensure you have permission to use the logos

## Fallback:
If logo files are not found, the component will display brand names as text instead.
